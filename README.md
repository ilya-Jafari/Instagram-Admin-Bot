# 📱 Instagram Admin Automation Bot

Ein leistungsstarkes Tool zur Automatisierung von Instagram-Interaktionen. Dieser Bot reagiert automatisch auf Kommentare unter bestimmten Reels mit Direktnachrichten (DMs) und öffentlichen Antworten.

## 🚀 Features
- **Keyword-Trigger:** Reagiert auf spezifische "WakeUpWords" (z.B. "INFO").
- **Reel-spezifisch:** Unterschiedliche Antworten für unterschiedliche Reels.
- **Dashboard:** Verwalte alle Regeln bequem über eine grafische Oberfläche (Streamlit).
- **Datenbank:** Schnelle und lokale Speicherung mit SQLite.
- **Public & Private:** Sendet DMs und antwortet gleichzeitig öffentlich auf Kommentare.

## 🛠 Tech-Stack
- **Sprache:** Python 3.x
- **Backend:** Flask (für Webhooks)
- **Frontend:** Streamlit (für das Dashboard)
- **Datenbank:** SQLite
- **API:** Instagram Graph API (Meta)
